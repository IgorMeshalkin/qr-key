# Пустой React-проект (Vite)

Минимальная заготовка React-приложения на Vite.

## Как запустить

1. Установите зависимости:
   ```bash
   npm install
   ```
   или
   ```bash
   yarn
   ```
   или
   ```bash
   pnpm install
   ```

2. Запустите dev-сервер:
   ```bash
   npm run dev
   ```

3. Откройте в браузере адрес, который покажет Vite (обычно http://localhost:5173/).

## Скрипты
- `npm run dev` — запуск дев-сервера
- `npm run build` — продакшн-сборка
- `npm run preview` — локальный предпросмотр собранной версии

## Структура
- `index.html` — HTML-входная точка
- `src/main.jsx` — инициализация React
- `src/App.jsx` — корневой компонент
- `src/index.css` — базовые стили
- `vite.config.js` — конфигурация Vite
- `package.json` — зависимости и скрипты


## Маршруты
В приложении настроены два роута:
- `/igor` — рендерит страницу `IgorPage.jsx`
- `/liza` — рендерит страницу `LizaPage.jsx`

На главном пути `/` настроен редирект на `/igor`. Для локальной разработки ничего дополнительно настраивать не нужно. Если будете деплоить как SPA на статическом хостинге, убедитесь, что включён fallback на `index.html` (например, настройка 404 -> index.html).
